# MyLanguage
My Old Russian language, which translates into my asm and then in binary code and excecute on my CPU:
https://github.com/Razeza/CPU

# Синтаксис

1. ХОТИВ...   - начало программы   
2. ИСПОВЕДАТИ - SCANF              
3. MEOW       - PRINTF 
4. АЖЕ        - IF 
5. ПОВЪДЪТИ   - PRINTF 
6. ^-^        - '{' и '}' 
7. СВЕРСТАН   - == 
8. ВЕРСТАТИ   - = 
9. ВЯЩЕ       - > 
10. ДОЛУ       - < 
11. УСПЕНИЕ    - конец программы 
12. ДОКАМЕСТ   - WHILE
